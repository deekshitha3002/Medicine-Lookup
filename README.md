# 💊 MedSearch

### *Search. Learn. Stay Informed.*

MedSearch is a web application that helps users quickly search for medicine information using trusted public APIs. It provides details such as medicine uses, side effects, warnings, dosage information, and AI-generated simplified explanations, making medical information easier to understand.

> **Disclaimer:** This application is for educational and informational purposes only. It is not a substitute for professional medical advice.

---

# 📌 Problem Statement

People often struggle to understand medicine information because medical terminology is complex and information is scattered across multiple websites. MedSearch solves this problem by providing reliable medicine information and AI-powered explanations in one place.

---

# ✨ Features

- 🔍 Search medicines by name
- 💊 View medicine uses
- ⚠️ Display common side effects
- 🚨 Show medicine warnings and precautions
- 🤖 AI-generated simplified explanations using Groq AI
- 🌐 Fetch real-time medicine data from the OpenFDA API
- 📱 Responsive and user-friendly interface
- ❌ Handles invalid or unavailable medicine searches gracefully
- 🛡️ Displays an informational medical disclaimer

---

# 🚀 Creative & Innovative Aspects

- Combines official medicine data from the OpenFDA API with AI-generated explanations.
- Simplifies complex medical information into easy-to-understand language.
- Provides users with reliable medicine information through a clean and responsive interface.
- Helps users quickly access important medicine details without browsing multiple websites.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/deekshitha3002/Medicine-Lookup.git
```

Navigate into the project

```bash
cd Medicine-Lookup
```

Install dependencies

```bash
npm install
```

Run the application

```bash
npm start
```

Open your browser and visit

```
http://localhost:3000
```
